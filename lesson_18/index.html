Связи между таблицами

Один к одному

One-to-one

Любому экземпляру сущности А соответствует только один экземпляр сущности В, и наоборот

Один ко многим

One-to-many

Любому экземпляру сущности А соответствует 0, 1 или несколько экземпляров сущности В, но любому экземпляру сущности В соответствует только один экземпляр сущности А

Многие ко многим

Many-to-many

Любому экземпляру сущности А соответствует 0, 1 или несколько экземпляров сущности В, и любому экземпляру сущности В соответствует 0, 1 или несколько экземпляров сущности А

Отношения в Laravel

                
                    <?php
                        namespace App;

                        use Illuminate\Database\Eloquent\Model;

                        class User extends Model
                        {
                          /**
                           * Получить запись с номером телефона пользователя.
                           */
                          public function phone()
                          {
                            return $this->hasOne('App\Phone');
                          }

                           /**
                           * Получить все сообщения пользователя
                           */
                          public function posts()
                          {
                            return $this->hasMany('App\Post');
                          }

                          /**
                           * Роли, принадлежащие пользователю.
                           */
                          public function roles()
                          {
                            return $this->belongsToMany('App\Role');
                          }
                        }
